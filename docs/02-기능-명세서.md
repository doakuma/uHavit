# 습관 생성 AI Agent 기능 명세서

## 📋 문서 정보
- **프로젝트명**: 습관 생성 AI Agent
- **버전**: 1.0
- **작성일**: 2026-01-XX

## 1. 기능 개요

본 문서는 습관 생성 AI Agent의 상세 기능 명세를 정의합니다. 각 기능은 MVP, Phase 2, Phase 3로 구분되어 있습니다.

---

## 2. MVP 기능 (Phase 1)

### 2.1 사용자 인증 및 프로필

#### 2.1.1 회원가입/로그인
- **기능**: 이메일 기반 회원가입 및 로그인
- **우선순위**: 필수
- **상세**:
  - 이메일, 비밀번호로 회원가입
  - 소셜 로그인 (Google) - 선택
  - 비밀번호 재설정 기능

#### 2.1.2 프로필 설정
- **기능**: 사용자 프로필 정보 설정
- **우선순위**: 필수
- **상세**:
  - 닉네임 설정
  - 프로필 이미지 업로드 (선택)
  - 알림 설정

### 2.2 습관 관리

#### 2.2.1 습관 생성
- **기능**: 새로운 습관 생성
- **우선순위**: 필수
- **입력 항목**:
  - 습관 이름 (필수, 최대 50자)
  - 목표 설명 (선택, 최대 200자)
  - 목표 수치 (선택, 예: 30분, 5km)
  - 빈도 (필수, 매일/주 N회/월 N회)
  - 카테고리 (선택, 운동/독서/건강/학습/업무/기타)
  - 알림 시간 (선택, 시간:분 형식)
  - 시작일 (기본값: 오늘)
- **AI 제안 기능**:
  - "AI가 습관 제안해줘" 버튼
  - 대화형 인터뷰를 통한 습관 제안
  - 제안된 습관을 바로 생성 가능

#### 2.2.2 습관 수정
- **기능**: 기존 습관 정보 수정
- **우선순위**: 필수
- **수정 가능 항목**: 생성 시 입력 항목과 동일

#### 2.2.3 습관 삭제
- **기능**: 습관 삭제
- **우선순위**: 필수
- **주의사항**: 삭제 전 확인 다이얼로그 표시

#### 2.2.4 습관 목록 조회
- **기능**: 사용자의 모든 습관 목록 조회
- **우선순위**: 필수
- **표시 정보**:
  - 습관 이름
  - 카테고리
  - 현재 연속 일수 (스트릭)
  - 전체 성공률
  - 오늘 완료 여부

### 2.3 습관 체크인

#### 2.3.1 완료 체크인
- **기능**: 습관 완료 기록
- **우선순위**: 필수
- **방식**:
  - 간단 체크 (완료/미완료)
  - 수치 입력 (예: 30분, 5km)
- **기록 시간**: 사용자가 선택 (기본값: 현재 시간)

#### 2.3.2 미완료 기록
- **기능**: 습관 미완료 기록
- **우선순위**: 필수
- **기능**: 체크인 화면에서 "완료하지 못함" 선택

#### 2.3.3 과거 날짜 수정
- **기능**: 과거 날짜의 체크인 수정
- **우선순위**: 필수
- **제한**: 최대 30일 전까지 수정 가능

### 2.4 AI 채팅

#### 2.4.1 AI와 대화
- **기능**: AI Agent와 텍스트 채팅
- **우선순위**: 필수
- **주요 대화 주제**:
  - 습관 계획 수립 조언
  - 진행 상황 피드백 요청
  - 동기부여 메시지 요청
  - 실패 원인 분석 요청
  - 습관 관련 질문

#### 2.4.2 컨텍스트 인식
- **기능**: AI가 사용자의 습관 데이터를 인식하여 맞춤형 조언 제공
- **우선순위**: 필수
- **인식 정보**:
  - 현재 습관 목록
  - 습관 수행 기록
  - 성공/실패 패턴
  - 최근 대화 내용

#### 2.4.3 빠른 선택지
- **기능**: 자주 사용하는 질문을 버튼으로 제공
- **우선순위**: 선택
- **예시**:
  - "오늘의 동기부여 메시지 줘"
  - "이번 주 진행 상황 분석해줘"
  - "습관 실패 원인 분석해줘"

### 2.5 통계 및 시각화

#### 2.5.1 대시보드
- **기능**: 홈 화면에 오늘의 습관 및 진행률 요약 표시
- **우선순위**: 필수
- **표시 정보**:
  - 오늘 완료해야 할 습관 목록
  - 오늘 완료율
  - 현재 연속 일수 (최고 스트릭)
  - 주간 성공률

#### 2.5.2 습관별 통계
- **기능**: 각 습관의 상세 통계 조회
- **우선순위**: 필수
- **통계 항목**:
  - 전체 성공률 (%)
  - 현재 연속 일수
  - 최고 연속 일수
  - 주간/월간 성공률
  - 성공률 트렌드 그래프

#### 2.5.3 캘린더 뷰 (히트맵)
- **기능**: 캘린더 형태로 습관 수행 기록 시각화
- **우선순위**: 필수
- **표시 방식**:
  - 완료: 초록색
  - 미완료: 회색
  - 오늘: 강조 표시
  - 색상 강도로 연속 일수 표현

#### 2.5.4 패턴 분석
- **기능**: 요일별, 시간대별 성공 패턴 분석
- **우선순위**: 선택
- **분석 항목**:
  - 요일별 성공률 히트맵
  - 시간대별 성공률 히트맵
  - 가장 성공률이 높은 요일/시간

### 2.6 알림

#### 2.6.1 습관 리마인더
- **기능**: 설정한 시간에 습관 수행 알림
- **우선순위**: 필수
- **방식**: 브라우저 알림 (웹)
- **설정**: 습관별로 알림 시간 설정 가능

#### 2.6.2 일일 요약 알림
- **기능**: 하루 종료 시 오늘의 성과 요약 알림
- **우선순위**: 선택
- **시간**: 사용자가 설정 (기본값: 오후 9시)

## 3. Phase 2 기능

### 3.1 고급 분석

#### 3.1.1 습관 간 상관관계 분석
- **기능**: 여러 습관 간의 상관관계 분석
- **예시**: "운동을 한 날은 수면 품질이 좋다"

#### 3.1.2 목표 달성 예측
- **기능**: 현재 패턴을 기반으로 목표 달성 가능성 예측
- **표시**: 성공 확률 (%)

### 3.2 확장된 기록 기능

#### 3.2.1 사진 첨부
- **기능**: 습관 수행 증거 사진 첨부
- **용도**: 운동 사진, 식사 사진 등

#### 3.2.2 일기 작성
- **기능**: 습관 수행 시 간단한 일기 작성
- **용도**: 느낌, 어려움, 성취감 기록

### 3.3 개인화

#### 3.3.1 대화 톤 커스터마이징
- **기능**: AI 대화 톤 설정
- **옵션**: 친근한 코치 / 전문 상담사 / 친구처럼

#### 3.3.2 다국어 지원
- **기능**: 한국어, 영어 지원
- **범위**: UI 및 AI 응답

### 3.4 자동 계획 조정

#### 3.4.1 실패 패턴 분석
- **기능**: 연속 실패 시 자동으로 원인 분석 및 계획 조정 제안
- **예시**: "주말에 실패율이 높네요. 주말 목표를 낮춰볼까요?"

## 4. Phase 3 기능

### 4.1 자동 추적

#### 4.1.1 앱 연동
- **기능**: 건강 앱, 운동 앱 등과 연동하여 자동 추적
- **예시**: Apple Health, Google Fit 연동

### 4.2 소셜 기능

#### 4.2.1 습관 공유
- **기능**: 습관 및 성과를 다른 사용자와 공유
- **방식**: 링크 공유, SNS 공유

#### 4.2.2 챌린지
- **기능**: 다른 사용자와 함께 챌린지 참여
- **예시**: "30일 운동 챌린지"

### 4.3 네이티브 모바일 앱

#### 4.3.1 iOS 앱
- **기능**: iOS 네이티브 앱 개발

#### 4.3.2 Android 앱
- **기능**: Android 네이티브 앱 개발

### 4.4 고급 AI 분석

#### 4.4.1 개인화된 습관 추천
- **기능**: 사용자 패턴을 분석하여 새로운 습관 추천

#### 4.4.2 장기 목표 설정 및 추적
- **기능**: 여러 습관을 조합한 장기 목표 설정 및 추적

## 6. 사용자 시나리오

### 6.1 신규 사용자 온보딩 시나리오
1. 회원가입/로그인
2. 온보딩 화면에서 "AI가 습관 제안해줘" 선택
3. AI와 대화하며 목표 및 라이프스타일 파악
4. AI가 제안한 습관 중 선택하여 생성
5. 첫 습관 체크인

### 6.2 일일 사용 시나리오
1. 앱 접속 → 대시보드에서 오늘의 습관 확인
2. 습관 수행 후 체크인
3. AI 채팅에서 오늘의 피드백 요청
4. 통계 화면에서 진행 상황 확인

### 6.3 습관 실패 시나리오
1. 습관 미완료 기록
2. AI 채팅에서 "왜 실패했는지 분석해줘" 요청
3. AI가 원인 분석 및 대안 제시
4. 필요시 습관 목표 조정

## 부록

### A. 에러 처리
- 네트워크 오류: 재시도 버튼 제공
- AI API 오류: 대체 메시지 표시
- 데이터 동기화 오류: 로컬 데이터 우선 표시

### B. 성능 요구사항
- 페이지 로딩 시간: 2초 이내
- AI 응답 시간: 5초 이내
- 데이터 동기화: 백그라운드 처리
