## 1. MVP 목표

MVP(Minimum Viable Product)의 목표는 **핵심 가치를 검증할 수 있는 최소 기능 세트**를 제공하는 것입니다.

### 1.1 핵심 가치
- 사용자가 습관을 생성하고 추적할 수 있음
- AI가 개인화된 조언을 제공할 수 있음
- 사용자가 자신의 습관 진행 상황을 시각적으로 확인할 수 있음

### 1.2 검증할 가설
- AI 기반 습관 형성 도우미가 사용자에게 가치를 제공하는가?
- 사용자가 AI와의 대화를 통해 습관을 효과적으로 형성할 수 있는가?
- 습관 추적 기능이 사용자의 동기부여에 도움이 되는가?

## 2. MVP 포함 기능

### 2.1 필수 기능 (Must Have)

#### ✅ 사용자 인증
- [x] 이메일 기반 회원가입/로그인
- [x] 비밀번호 재설정
- [x] 로그아웃

#### ✅ 습관 관리
- [x] 습관 생성 (이름, 목표, 빈도, 카테고리)
- [x] 습관 수정
- [x] 습관 삭제
- [x] 습관 목록 조회
- [x] 습관 활성화/비활성화

#### ✅ 습관 체크인
- [x] 완료 체크인 (간단 체크)
- [x] 수치 입력 체크인 (예: 30분, 5km)
- [x] 미완료 기록
- [x] 과거 날짜 수정 (최대 30일 전)

#### ✅ AI 채팅
- [x] AI와 텍스트 채팅
- [x] 습관 계획 수립 조언 요청
- [x] 진행 상황 피드백 요청
- [x] 동기부여 메시지 요청
- [x] 사용자 컨텍스트 인식 (습관 데이터 기반)

#### ✅ 기본 통계
- [x] 대시보드 (오늘의 습관, 진행률 요약)
- [x] 습관별 성공률
- [x] 현재 연속 일수 (스트릭)
- [x] 주간/월간 성공률
- [x] 성공률 트렌드 그래프

#### ✅ 캘린더 뷰
- [x] 히트맵 형태의 캘린더
- [x] 완료/미완료 시각화
- [x] 연속 일수 표시

#### ✅ 알림
- [x] 습관별 리마인더 알림 (브라우저 알림)
- [x] 알림 시간 설정

### 2.2 선택 기능 (Nice to Have)

#### ⚠️ 패턴 분석
- [ ] 요일별 성공률 히트맵
- [ ] 시간대별 성공률 히트맵
- [ ] 가장 성공률이 높은 요일/시간 분석

#### ⚠️ 빠른 선택지
- [ ] AI 채팅에서 자주 사용하는 질문 버튼
- [ ] "오늘의 동기부여 메시지" 빠른 버튼

## 3. MVP 제외 기능 (Phase 2 이후)

다음 기능들은 MVP에서 제외되며, Phase 2 또는 Phase 3에서 구현됩니다.

### 3.1 Phase 2 기능

- 목표 달성 예측
- 사진 첨부

### 3.2 Phase 3 기능

- 고급 AI 분석

## 4. 기술 스택 (MVP)

### 4.1 프론트엔드
- React 18+ (TypeScript)
- Vite (빌드 도구)
- React Router (라우팅)
- Zustand 또는 React Query (상태 관리)
- CSS Modules (스타일링)

### 4.2 백엔드
- Supabase (BaaS)
  - PostgreSQL 데이터베이스
  - 인증 (Auth)
  - 실시간 구독 (선택)

### 4.3 AI
- OpenAI GPT API (gpt-4o-mini)
- Supabase Edge Functions로 래핑

### 4.4 배포
- Vercel 또는 Netlify (프론트엔드)
- Supabase (백엔드)

## 12. 성공 기준 (MVP)

### 12.1 기능적 기준
- ✅ 모든 필수 기능이 정상 작동

- ✅ AI 채팅이 의미 있는 응답 제공

- ✅ AI 응답 시간 5초 이내
- ✅ 모바일에서 사용 가능 (반응형)

- ✅ 베타 테스터 10명 이상
