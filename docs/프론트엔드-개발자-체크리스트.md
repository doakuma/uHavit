## 1. 프로젝트 설정 및 환경

### ✅ 확인 완료 사항
- [x] **패키지 관리자**: `pnpm` 사용 (v10.28.0)
- [x] **Node.js 버전**: 18.0.0 이상 요구
- [x] **빌드 도구**: Vite 5.0.0 사용
- [x] **React 버전**: 18.2.0
- [x] **경로 Alias 설정**: `@/`, `@components/`, `@pages/` 등 설정 완료

### ⚠️ 확인 필요 사항
- [ ] **환경 변수 파일**: `.env.local` 파일 존재 여부 확인
  - `VITE_SUPABASE_URL` 설정 확인
  - `VITE_SUPABASE_ANON_KEY` 설정 확인
  - `VITE_OPENAI_API_KEY` 설정 확인 (AI 기능 사용 시)
- [ ] **환경 변수 검증**: 개발 환경에서 환경 변수 누락 시 경고 표시 확인
- [ ] **더미 모드**: 환경 변수 없을 때 더미 모드로 동작하는지 확인

### 📝 권장 사항
- [ ] `.env.example` 파일 생성하여 필수 환경 변수 문서화
- [ ] 환경 변수 검증 로직 추가 (빌드 시점 체크)

## 3. 상태 관리

### ✅ 확인 완료 사항
- [x] **React Query**: TanStack Query v5 사용
- [x] **Query 설정**: 기본 staleTime 5분, refetchOnWindowFocus 비활성화
- [x] **Custom Hooks**: `useAuth`, `useHabits`, `useCheckins`, `useAIChat` 구현
- [x] **캐시 무효화**: Mutation 성공 시 적절한 쿼리 무효화

### ⚠️ 확인 필요 사항
- [ ] **로컬 상태 관리**: useState 사용이 적절한지 확인
- [ ] **전역 상태**: Context API 필요 여부 검토
- [ ] **Optimistic Updates**: 사용자 경험 개선을 위한 낙관적 업데이트 적용 여부
- [ ] **쿼리 키 관리**: 쿼리 키 일관성 확인

### 📝 권장 사항
- [ ] **Optimistic Updates** 적용 (예: 체크인 완료 시 즉시 UI 업데이트)

  };
